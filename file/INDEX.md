# file/ — 文件操作、归档与传输

## 命令清单

| 命令 | 文件 | 说明 |
|------|------|------|
| `ls` | [ ] | 列出目录内容 |
| `cp` | [ ] | 复制文件或目录 |
| `mv` | [ ] | 移动/重命名文件 |
| `rm` | [ ] | 删除文件或目录 |
| `find` | [ ] | 递归查找文件 |
| `tree` | [ ] | 树状显示目录结构 |
| `stat` | [ ] | 查看文件属性 |
| `touch` | [ ] | 创建空文件/更新时间戳 |
| `mkdir` | [ ] | 创建目录 |
| `ln` | [ ] | 创建硬链接/软链接 |
| `tar` | [ ] | 打包归档 |
| `gzip` / `gunzip` | [ ] | 压缩/解压 .gz |
| `zip` / `unzip` | [ ] | 压缩/解压 .zip |
| `xz` | [ ] | 高压缩率工具 |
| `rsync` | [ ] | 远程/本地同步 |
| `scp` | [ ] | 远程安全拷贝 |
