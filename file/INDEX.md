# file/ — 文件操作、归档与传输

## 命令清单

| 命令 | 文件 | 说明 |
|------|------|------|
| `ls` | [x] | 列出目录内容 |
| `cp` | [x] | 复制文件或目录 |
| `mv` | [x] | 移动/重命名文件 |
| `rm` | [x] | 删除文件或目录 |
| `find` | [x] | 递归查找文件 |
| `tree` | [x] | 树状显示目录结构 |
| `stat` | [x] | 查看文件属性 |
| `touch` | [x] | 创建空文件/更新时间戳 |
| `mkdir` | [x] | 创建目录 |
| `ln` | [x] | 创建硬链接/软链接 |
| `tar` | [x] | 打包归档 |
| `gzip` / `gunzip` | [x] | 压缩/解压 .gz |
| `zip` / `unzip` | [x] | 压缩/解压 .zip |
| `xz` | [x] | 高压缩率工具 |
| `rsync` | [x] | 远程/本地同步 |
| `scp` | [x] | 远程安全拷贝 |
