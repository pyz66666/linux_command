# system/ — 系统信息与管理

## 命令清单

| 命令 | 文件 | 说明 |
|------|------|------|
| `uname` | [ ] | 系统内核信息 |
| `lscpu` | [ ] | CPU 信息 |
| `lspci` | [ ] | PCI 设备列表 |
| `lsmod` | [ ] | 已加载内核模块 |
| `lsusb` | [ ] | USB 设备列表 |
| `free` | [ ] | 内存用量 |
| `uptime` | [ ] | 运行时间和负载 |
| `journalctl` | [ ] | systemd 日志查询 |
| `dmidecode` | [ ] | 硬件信息（DMI） |
| `hostnamectl` | [ ] | 主机名管理 |
| `timedatectl` | [ ] | 时区和时间管理 |
| `sysctl` | [ ] | 内核参数读写 |
| `modprobe` | [ ] | 内核模块加载 |
