# system/ — 系统信息与管理

## 命令清单

| 命令 | 文件 | 说明 |
|------|------|------|
| `uname` | [x] | 系统内核信息 |
| `free` | [x] | 内存用量 |
| `journalctl` | [x] | systemd 日志查询 |
| `lscpu` | [x] | CPU 信息 |
| `uptime` | [x] | 运行时间和负载 |
| `lspci` | [x] | PCI 设备列表 |
| `dmidecode` | [x] | 硬件信息（DMI） |
| `lsmod` | [x] | 已加载内核模块 |
| `sysctl` | [x] | 内核参数读写 |
| `timedatectl` | [x] | 时区和时间管理 |
| `hostnamectl` | [x] | 主机名管理 |
| `lsusb` | [x] | USB 设备列表 |
| `modprobe` | [x] | 内核模块加载 |
