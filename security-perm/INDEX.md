# security-perm/ — 权限控制与安全

## 命令清单

| 命令 | 文件 | 说明 |
|------|------|------|
| `chmod` | [ ] | 修改文件权限 |
| `chown` | [ ] | 修改文件所有者 |
| `umask` | [ ] | 默认权限掩码 |
| `sudo` | [ ] | 以 root 身份执行 |
| `su` | [ ] | 切换用户 |
| `useradd` / `userdel` | [ ] | 用户创建/删除 |
| `usermod` | [ ] | 用户属性修改 |
| `passwd` | [ ] | 密码管理 |
| `groupadd` / `groupmod` | [ ] | 用户组管理 |
| `setfacl` / `getfacl` | [ ] | ACL 访问控制 |
| `iptables` | [ ] | 防火墙规则 |
| `firewalld` | [ ] | 动态防火墙 |
| `ssh-keygen` | [ ] | SSH 密钥生成 |
| `openssl` | [ ] | 加密与证书工具 |
