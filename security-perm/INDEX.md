# security-perm/ — 权限控制与安全

## 命令清单

| 命令 | 文件 | 说明 |
|------|------|------|
| `chmod` | [x] | 修改文件权限 |
| `chown` | [x] | 修改文件所有者 |
| `sudo` | [x] | 以 root 身份执行 |
| `ssh-keygen` | [x] | SSH 密钥生成 |
| `passwd` | [x] | 密码管理 |
| `iptables` | [x] | 防火墙规则 |
| `useradd` / `userdel` | [x] | 用户创建/删除 |
| `usermod` | [x] | 用户属性修改 |
| `umask` | [x] | 默认权限掩码 |
| `openssl` | [x] | 加密与证书工具 |
| `su` | [x] | 切换用户 |
| `setfacl` / `getfacl` | [x] | ACL 访问控制 |
| `groupadd` / `groupmod` | [x] | 用户组管理 |
| `firewalld` | [x] | 动态防火墙 |
