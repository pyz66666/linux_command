# process/ — 进程管理与服务

## 命令清单

| 命令 | 文件 | 说明 |
|------|------|------|
| `ps` | [ ] | 查看进程状态 |
| `top` / `htop` | [ ] | 实时进程监控 |
| `kill` / `killall` | [ ] | 终止进程 |
| `pgrep` / `pkill` | [ ] | 按名称查找/终止进程 |
| `nice` / `renice` | [ ] | 调整进程优先级 |
| `nohup` | [ ] | 忽略 HUP 信号运行 |
| `jobs` / `bg` / `fg` | [ ] | 作业控制 |
| `systemctl` | [ ] | 管理 systemd 服务 |
| `crontab` | [ ] | 定时任务 |
| `lsof` | [ ] | 列出打开的文件 |
