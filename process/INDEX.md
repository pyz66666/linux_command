# process/ — 进程管理与服务

## 命令清单

| 命令 | 文件 | 说明 |
|------|------|------|
| `ps` | [x] | 查看进程状态 |
| `top` / `htop` | [x] | 实时进程监控 |
| `kill` / `killall` | [x] | 终止进程 |
| `pgrep` / `pkill` | [x] | 按名称查找/终止进程 |
| `nice` / `renice` | [x] | 调整进程优先级 |
| `nohup` | [x] | 忽略 HUP 信号运行 |
| `jobs` / `bg` / `fg` | [x] | 作业控制 |
| `systemctl` | [x] | 管理 systemd 服务 |
| `crontab` | [x] | 定时任务 |
| `lsof` | [x] | 列出打开的文件 |
