# network/ — 网络诊断与工具

## 命令清单

| 命令 | 文件 | 说明 |
|------|------|------|
| `curl` | [x] | HTTP 客户端/数据传输 |
| `wget` | [x] | 文件下载 |
| `ss` | [x] | 套接字统计（替代 netstat） |
| `ping` | [x] | ICMP 连通性测试 |
| `traceroute` / `mtr` | [x] | 路由追踪 |
| `tcpdump` | [x] | 抓包分析 |
| `nmap` | [x] | 端口扫描 |
| `dig` / `nslookup` | [x] | DNS 查询 |
| `nc` | [x] | 网络瑞士军刀 |
| `iperf` / `iperf3` | [x] | 网络带宽测试 |
| `ip` | [x] | 网络配置（替代 ifconfig） |
| `iptraf-ng` | [x] | 实时网络流量监控 |
