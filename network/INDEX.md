# network/ — 网络诊断与工具

## 命令清单

| 命令 | 文件 | 说明 |
|------|------|------|
| `curl` | [ ] | HTTP 客户端/数据传输 |
| `wget` | [ ] | 文件下载 |
| `ss` | [ ] | 套接字统计（替代 netstat） |
| `ping` | [ ] | ICMP 连通性测试 |
| `traceroute` / `mtr` | [ ] | 路由追踪 |
| `tcpdump` | [ ] | 抓包分析 |
| `nmap` | [ ] | 端口扫描 |
| `dig` / `nslookup` | [ ] | DNS 查询 |
| `nc` | [ ] | 网络瑞士军刀 |
| `iperf` / `iperf3` | [ ] | 网络带宽测试 |
| `ip` | [ ] | 网络配置（替代 ifconfig） |
| `iptraf-ng` | [ ] | 实时网络流量监控 |
