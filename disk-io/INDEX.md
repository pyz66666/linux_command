# disk-io/ — 磁盘、存储与 I/O

## 命令清单

| 命令 | 文件 | 说明 |
|------|------|------|
| `df` | [x] | 磁盘空间使用概览 |
| `du` | [x] | 目录/文件大小统计 |
| `lsblk` | [x] | 列出块设备 |
| `mount` / `umount` | [x] | 挂载/卸载文件系统 |
| `fdisk` / `parted` | [x] | 磁盘分区工具 |
| `dd` | [x] | 底层数据读写/复制 |
| `iostat` | [x] | I/O 统计 |
| `iotop` | [x] | 实时 I/O 监控 |
| `fio` | [x] | I/O 性能压测 |
| `iozone` | [x] | 文件系统性能测试 |
| `lvm` | [x] | 逻辑卷管理 |
| `fsck` | [x] | 文件系统检查修复 |
