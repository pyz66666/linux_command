# disk-io/ — 磁盘、存储与 I/O

| 命令 | 文件 | 说明 |
|------|------|------|
| `df` | [ ] | 磁盘空间使用概览 |
| `du` | [ ] | 目录/文件大小统计 |
| `lsblk` | [ ] | 列出块设备 |
| `mount` / `umount` | [ ] | 挂载/卸载文件系统 |
| `fdisk` / `parted` | [ ] | 磁盘分区工具 |
| `dd` | [ ] | 底层数据读写/复制 |
| `iostat` | [ ] | I/O 统计 |
| `iotop` | [ ] | 实时 I/O 监控 |
| `fio` | [ ] | I/O 性能压测 |
| `iozone` | [ ] | 文件系统性能测试 |
| `lvm` | [ ] | 逻辑卷管理 |
| `fsck` | [ ] | 文件系统检查修复 |
