# perf-debug/ — 性能分析与调试

## 命令清单

| 命令 | 文件 | 说明 |
|------|------|------|
| `dmesg` | [x] | 内核环形缓冲区日志 |
| `strace` | [x] | 跟踪系统调用 |
| `perf` | [x] | Linux 性能分析器 |
| `vmstat` | [x] | 虚拟内存统计 |
| `sar` | [x] | 系统活动报告 |
| `gdb` | [x] | GNU 调试器 |
| `pidstat` | [x] | 进程级资源统计 |
| `bpftrace` | [x] | 动态追踪工具 |
| `ltrace` | [x] | 跟踪库函数调用 |
| `slabtop` | [x] | 内核 slab 缓存监控 |
