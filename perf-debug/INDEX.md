# perf-debug/ — 性能分析与调试

## 命令清单

| 命令 | 文件 | 说明 |
|------|------|------|
| `perf` | [ ] | Linux 性能分析器 |
| `strace` | [ ] | 跟踪系统调用 |
| `ltrace` | [ ] | 跟踪库函数调用 |
| `vmstat` | [ ] | 虚拟内存统计 |
| `sar` | [ ] | 系统活动报告 |
| `dmesg` | [ ] | 内核环形缓冲区日志 |
| `bpftrace` | [ ] | 动态追踪工具 |
| `gdb` | [ ] | GNU 调试器 |
| `pidstat` | [ ] | 进程级资源统计 |
| `slabtop` | [ ] | 内核 slab 缓存监控 |
